# startup

## Пока что не готов

Статический лендинг для разработчика или небольшой команды, которая берёт веб-приложение целиком: от первого созвона и технического контура до кода, production-деплоя и поддержки после релиза. Это одностраничный сайт без сборщика и без backend-части.

## Что есть на странице

- hero-блок с canvas-фоном, scroll progress и акцентом на backend, инфраструктуру и релизы;
- блок услуг с интерактивными табами: `Discovery`, `Architecture`, `Build`, `Deploy`, `AI`, `Support`;
- секция процесса из 7 этапов с анимируемой timeline;
- портфолио-блок с живым кейсом `AiNoCraft` и местом под следующий реальный запуск;
- секция стека с frontend, backend и DevOps-направлением;
- блок команды с двумя ролями;
- контактная форма без серверной отправки: она открывает `mailto:`-черновик письма.

## Особенности реализации

Страница собрана на чистом HTML, CSS и JavaScript.

Основные интерактивные элементы:

- переключение языка `RU / EN` через встроенный `i18n`-словарь в `script.js`;
- переключение темы `dark / light` с сохранением выбора в `localStorage`;
- анимированный `flow-field` canvas на фоне;
- reveal-анимации секций при скролле;
- tilt-эффект карточек при наведении;
- анимация строк кода в hero-блоке;
- переключатель сервисов по табам;
- индикатор прогресса для секции процесса;
- контактная форма через `mailto:hello@startup.dev`.

В коде учтён `prefers-reduced-motion`: если у пользователя отключены лишние анимации на уровне системы, часть эффектов не запускается.

## Структура проекта

```text
startup/
├── index.html
├── style.css
└── script.js
```

## Файлы проекта

- `index.html` — разметка лендинга, секций и основных карточек;
- `style.css` — визуальный стиль, темы, layout, анимации и адаптивность;
- `script.js` — i18n, theme toggle, scroll UI, canvas-анимация, tabs, timeline и логика контактной формы.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (`IBM Plex Sans`, `IBM Plex Mono`, `Sora`)
